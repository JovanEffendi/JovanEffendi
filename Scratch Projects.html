Doodle Games - MAZE

A Maze Game in Scratch

A maze game in Scratch is an interactive experience where players guide a character (sprite) through a maze to reach a goal without touching the walls. The backdrop serves as the maze. Players use keyboard inputs, like the arrow keys, to move the sprite up, down, left, or right.

Key Features:

  • Collision Detection: If the sprite touches a wall, it might lose a life, restart, or face a penalty.
  • Levels: Completing the maze can lead to more challenging levels with narrower paths or additional obstacles.
  • Enhancements: Games may include timers, scoring systems, or sound effects to make the experience more engaging.

Doodle Games - Clicker

A Clicker Game in Scratch

This game is very straightforward, with only 1 objective. To click as much as possible in a certain amount of time. Also, IT NEVER ENDS! This game was to be used as a foundation for many other games.

Key Features:

  • Button:Click the button to get points.
  • Points System:The points will increase with each button press and will reset after stopping the game.

Doodle Games - Hoop Challenge

A Mini-Basketball in Scratch

This basketball game is quite different than a real basketball game. Instead of the person or the basketball moving, the hoop will move randomly across the screen. Isn't that confusing? Try the game to find out!

  • Moving Hoop:The Basketball hoop will move in random directions across the Scratch Window.
  • Points System:The score will increase with each ball that successfully goes in the hoop.
  • Shooting:To Shoot, you have to click the spacebar.
